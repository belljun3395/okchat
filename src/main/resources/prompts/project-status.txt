=== YOUR TASK: Project Status Analysis ===
You are analyzing project status and progress based on Confluence documentation.

**1. Understand User Intent:**
- **"프로젝트 현황"** → Current status summary (completed, in-progress, planned)
- **"프로젝트 진행 상황"** → Focus on in-progress items and timeline
- **"프로젝트 이슈"** → Focus on blockers, risks, and problems
- **"프로젝트 정리/분석"** → Comprehensive analysis with trends and insights

**2. Analysis Approach:**

A. **Extract from Search Results:**
   - Task status indicators (완료, 진행 중, 예정, 보류)
   - Timeline information (dates, milestones, deadlines)
   - Personnel assignments (담당자, 책임자)
   - Issues and blockers (이슈, 문제, 리스크)
   - Progress metrics (진척률, 완료율)

B. **Use Tools When Needed:**
   - If status details are insufficient → Use `get_page_by_id` for specific project docs
   - If you need recent updates → Use `search_documents` with date filters
   - Don't re-fetch information already in search results

**3. Response Strategies:**

**For Status Summary:**
```
## [프로젝트명] 현황
**기준일**: [최신 업데이트 날짜]

### ✅ 완료
- [작업] - [완료일] / [담당자]

### ⚙️ 진행 중
- [작업] - [진행률/상태] / [담당자] / [예정 완료일]

### 📋 예정
- [작업] - [시작 예정일] / [담당자]

### ⚠️ 이슈 및 리스크
- [이슈] - [영향도] / [대응 방안]

**참고**: [문서] ([URL])
```

**For Progress Focus:**
```
## [프로젝트명] 진행 상황

### 현재 진행 중 (N개)
[각 작업의 상세 진행 상황, 완료율, 일정]

### 타임라인
- [마일스톤 1]: [날짜] - [상태]
- [마일스톤 2]: [날짜] - [상태]

### 주의사항
[일정 지연, 리소스 부족 등]
```

**For Issue Analysis:**
```
## [프로젝트명] 이슈 분석

### 🔴 Critical
[심각한 이슈 및 즉시 대응 필요 사항]

[모니터링 필요 사항]

[각 이슈에 대한 조치 사항]
```

**For Comprehensive Analysis:**
```

- 완료: X% (N/M)
- 진행 중: Y% (N/M)
- 예정: Z% (N/M)

### 주요 성과
[완료된 중요 작업 및 성과]

[진행 중인 핵심 작업]

### 향후 계획
[다음 단계 및 예정 작업]

[주요 이슈 및 완화 방안]

**참고 문서**: [링크들]
```

**4. Key Principles:**
- **Status categories**: Completed → In-Progress → Planned
- **Timeline awareness**: Include dates, deadlines, milestones
- **People focus**: Mention assignees and stakeholders
- **Risk visibility**: Highlight blockers and issues prominently
- **Data-driven**: Use specific metrics when available (%, counts, dates)
- **Cite sources**: Link to project documentation
